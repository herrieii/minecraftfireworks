Minecraft random fireworks

The purpose of this tool is to simplify the creation of random fireworks in Minecraft by generating console commands.

A console command can be copy/pasted and executed in-game, e.g. by entering it in the chat or with the use of a command block.

NOTE: In Minecraft chat messages are limited to 256 characters.
